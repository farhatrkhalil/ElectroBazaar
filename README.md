# ElectroBazaar: Full-Stack E-Commerce Engine

**ElectroBazaar** is a high-performance e-commerce platform engineered with a decoupled architecture. It leverages a React-based frontend and a Firebase-as-a-Service (FaaS) backend to provide real-time data synchronization and scalable inventory management.
---

## üõ† Engineering Architecture

* **Real-time Synchronization:** Utilizes Firebase Realtime Database/Firestore for instantaneous UI updates across client sessions.
* **Administrative Logic:** Integrated a secure Admin Dashboard for full CRUD (Create, Read, Update, Delete) operations on the product catalog.
* **State & Navigation:** Implemented complex routing via React Router and managed user state for personalized shopping experiences.
* **Search & Filtering:** Engineered optimized search algorithms to query large datasets of computing devices and peripherals.

## üöÄ Technical Stack

* **Frontend:** React.js (Component-based architecture)
* **Backend/BaaS:** Firebase (Authentication, Database, and Hosting)
* **Styling:** Modular CSS / Tailwind 
* **Version Control:** Git/GitHub

### üîπ Client-Side Experience

### üîπ Infrastructure & Management
